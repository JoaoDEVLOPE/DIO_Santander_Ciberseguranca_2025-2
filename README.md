# DIO_Santander_Ciberseguranca_2025-2
Continuação do Bootcamp de Cibersegurança, com foco nos diferentes tipos de malwares. Nesta etapa, destacamos dois dos mais conhecidos, ransomware e keylogger e utilizamos Python para demonstrar sua execução. Ao final, apresentamos dicas práticas sobre como se proteger e evitar esses tipos de ameaças. 

Ransomware: foi preparado um ambiente de teste controlado para a criação de um código simples, desenvolvido exclusivamente para fins éticos e de estudo. Em seguida, foi implementado um código para a descriptografia, acompanhado por uma imagem demonstrando a execução dessa etapa após a primeira parte

Para manter o experimento discreto durante a execução, o arquivo foi renomeado. Criamos ainda um e-mail específico para tratar do envio/recebimento de mensagens durante os testes.

Prevenção e detecção no dia a dia:
Mantenha antivírus e firewall sempre atualizados. Um antivírus moderno não se limita a checar listas de arquivos maliciosos, ele também analisa o comportamento dos programas, reconhece padrões de ataque e bloqueia atividades suspeitas. Um firewall bem configurado pode, por exemplo, impedir que processos desconhecidos enviem e-mails via SMTP. A detecção por comportamento, que identifica rapidamente ações suspeitas como criptografia em massa de arquivos, já se tornou prática comum na cibersegurança.

Outro ponto importante é a consciência do usuário. É fundamental ter cuidado com anexos, links disfarçados de confiáveis, PDFs falsos ou qualquer material que possa conter malware, um tipo de ataque conhecido como Engenharia Social.

Por fim, sempre utilize ambientes isolados para testes, como máquinas virtuais, ao rodar códigos desconhecidos. Isso cria uma barreira entre o ambiente de teste e o sistema principal, evitando riscos ao computador real.
