# DIO_Santander_Ciberseguranca_2025-2
Continuação do Bootcamp de Cibersegurança, com foco nos diferentes tipos de malwares. Nesta etapa, destacamos dois dos mais conhecidos, ransomware e keylogger e utilizamos Python para demonstrar sua execução. Ao final, apresentamos dicas práticas sobre como se proteger e evitar esses tipos de ameaças. 

Ransomware: foi preparado um ambiente de teste controlado para a criação de um código simples, desenvolvido exclusivamente para fins éticos e de estudo. Em seguida, foi implementado um código para a descriptografia, acompanhado por uma imagem demonstrando a execução dessa etapa após a primeira parte

![image alt](https://github.com/JoaoDEVLOPE/DIO_Santander_Ciberseguranca_2025-2/blob/bf0e68f70b21fa2936a38d9f33ba860952810abf/Descriptografia.jpeg)

Keylogger: em ambiente de testes controlado, desenvolvemos um código para fins éticos e educativos. O programa registrava as teclas digitadas e gravava os dados em um arquivo log.txt. Para manter o experimento discreto durante a execução, o arquivo foi renomeado. Criamos ainda um e-mail específico para tratar do envio/recebimento de mensagens durante os testes.
